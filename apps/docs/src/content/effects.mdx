# Effects

Every timeline entry is one **effect**, discriminated by `action`. They fall into
five lanes: camera, navigation, cursor, interaction and timing.

## Reference

Shared optional fields on every effect: `frameId`, `delayBefore`, `delayAfter`,
`note`. `easing` is one of `linear · ease-in · ease-out · ease-in-out · spring`.

| Action | Key fields | Default duration |
| --- | --- | --- |
| `click` | `frameId`, `showRipple`, `soundProfile` | ~500ms |
| `type-down` | `frameId`, `text`, `typingSpeed`, `mistakes` | `text.length × speed` |
| `drag-and-drop` | `frameId`, `destFrameId`/`destX,destY` | 800ms |
| `shake` | `frameId`, `intensity` | 500ms |
| `zoom-in` | `frameId?`, `scale`, `transformOrigin` | 1200ms |
| `zoom-out` | — | 900ms |
| `scroll-up` / `scroll-down` | `distance` | 700ms |
| `cursor-move` | `destFrameId`/`destX,destY`, `curve` | 700ms |
| `highlight` | `frameId`, `dimOpacity`, `padding` | 1500ms |
| `wait` | `duration` | 1000ms |

All are strongly typed with Zod; the [timesheet](/timesheet) schema rejects an
unplayable sheet before a single frame is recorded.
